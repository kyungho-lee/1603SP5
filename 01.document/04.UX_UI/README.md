# Project review

## Goal

- ITCH.io 에 완성된 챕터를 런칭한다.
- Unreal Engine의 기능의 장점을 최대한 활용 (물리엔진,그래픽연출)
- 마이크로 애자일개발방향으로 더 작은 단위의 스크럼을 이용해 빠른 피드백과 수정을 한다.

## Develop Flow

1. Preproduction 
   1. KickOff
   2. 아이디어 회의
   3. 컨셉회의 (시각화)
   4. 프로토타이핑
   5. 자료조사 
2. Production
   1. 컨셉기획
   2. 개발계획
   3. 시스템기획
   4. 기능개발
   5. 리소스제작(2차가공)
3. Postproduction
